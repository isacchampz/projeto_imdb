# 🎬 Projeto IMDb - Análise de Dados com Power BI

## 📌 Sobre o Projeto

Este projeto tem como objetivo analisar e organizar dados de filmes do IMDb utilizando o Power BI, criando dashboards interativos para facilitar a visualização das informações.

## 📊 Dados Utilizados

Os dados utilizados neste projeto são provenientes do IMDb (Internet Movie Database), contendo informações como:

* Título dos filmes
* Ano de lançamento
* Duração
* Gênero
* Avaliação (rating)
* Orçamento
* Bilheteria
* Elenco

## ⚙️ Ferramentas Utilizadas

* Power BI
* Excel/CSV

## 📈 Análises Realizadas

* O gênero com maior receita bruta
